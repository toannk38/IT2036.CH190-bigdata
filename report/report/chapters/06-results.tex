\section{Weighted Scoring System}

Hệ thống tính điểm tổng hợp dựa trên:
\begin{itemize}
\item Technical indicators (40\%)
\item Sentiment score (30\%)
\item News impact (20\%)
\item Market trend (10\%)
\end{itemize}

\section{Alert Generation}

Hệ thống tự động tạo alerts khi:
\begin{itemize}
\item Điểm tổng hợp > 0.7 (BUY signal)
\item Điểm tổng hợp < 0.3 (SELL signal)
\item Volatility bất thường
\item Breaking news impact
\end{itemize}
